# Cratex — Платформа онлайн-курсов

Этот проект представляет собой верстку главной страницы для платформы по продаже и прохождению онлайн-курсов. Это учебный проект, сфокусированный на адаптивности и структуре веб-страницы.

## 🛠 Технологии
* HTML5
* CSS3

## 🚀 Как запустить
Для просмотра проекта не требуется установка дополнительных программ или серверов:
1. Скачайте репозиторий или склонируйте его.
2. Откройте файл `index.html` в любом современном браузере.

## 📌 Статус проекта
Проект находится в активной разработке. На данный момент реализована верстка главной страницы. В планах — добавление интерактивных элементов и адаптация под мобильные устройства.
